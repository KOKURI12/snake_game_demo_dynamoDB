---

## 最初に確認するファイル

作業開始時は、まず以下を確認してください。

1. `AGENTS.md`
2. `PROJECT_CONTEXT.md`
3. プロジェクトのエントリーポイント（例：package.json / pyproject.toml / go.mod 等）
4. `README.md`
5. 依頼に関連するソースコード

---

## Claude Code 作業ルール

- 変更前に短い作業計画を提示してください。
- 依頼と関係ないファイルは変更しないでください。
- 大規模なディレクトリ変更は事前確認なしに行わないでください。
- 新規依存パッケージの追加は必ず事前確認してください。
- AWS / DB / 認証情報に関わる変更は特に慎重に扱ってください。

---

## AWS 関連の注意

以下は事前確認なしに変更しないでください。

- DynamoDB / RDS 等のデータベース
- Lambda / ECS / EC2 等のコンピュートリソース
- API Gateway / CloudFront 等のフロントエンド
- IAM Role / IAM Policy
- セキュリティグループ / VPC 設定
- デプロイ設定（SAM / CDK / Terraform 等）
- 課金が発生する可能性のある設定
